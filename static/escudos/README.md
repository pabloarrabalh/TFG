# Escudos de Equipos

Carpeta centralizada para almacenar las imágenes de los escudos de todos los equipos de La Liga.

## Estructura:
```
escudos/
├── almeria.png
├── athletic.png
├── atletico.png
├── barcelona.png
├── betis.png
├── cadiz.png
├── celta.png
├── girona.png
├── granada.png
├── getafe.png
├── laspalmas.png
├── leganes.png
├── mallorca.png
├── osasuna.png
├── rayo.png
├── realsociedad.png
├── realmadrid.png
├── sevilla.png
├── sociedad.png
├── valladolid.png
├── valencia.png
└── villarreal.png
```

## Notas:
- Los nombres de archivo deben coincidir con los nombres de equipos en minúsculas (sin espacios ni tildes)
- Se recomienda usar formato PNG con fondo transparente
- Tamaño recomendado: 150x150px o 200x200px
- Los mismos escudos se usan para todas las temporadas (mismo equipo = mismo escudo)
